Important HTML tags

This page will act as a reference for some common tags. In the process of making it, we'll learn how to make tables via table and related tags.

The table below do not include all HTML tags, but they do list many of the important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>nice skills</em> nice skills
strong strong make strong text <strong>well hello</strong> well hello
a anchor make hyperlink <a href="https://runescape.com/">do you dare?</a> do you dare?
img image include an image <img src="https://tinyurl.com/auher8iw3h" alt="pikachu">
